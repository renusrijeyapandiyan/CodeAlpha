# CodeAlpha Project

This repository contains multiple Python projects for data analysis and machine learning.

## Projects Included

1. **Car Price Prediction with Machine Learning**
   - Predict car prices based on dataset features
   - Files: `car_price_prediction.py`, `car data.csv`

2. **Iris Flower Classification**
   - Classify Iris flowers using ML algorithms
   - Files: `iris_classification.py`, `Iris.csv`

3. **Sales Prediction using Python**
   - Predict sales using advertising data
   - Files: `sales_prediction.py`, `Advertising.csv`, `Sales_Predictions.csv`

4. **Unemployment Analysis with Python**
   - Analyze unemployment trends in India
   - Files: `unemployment_analysis.py`, datasets
